# R · Portfolio (Vite + React)

Dark editorial portfolio for a recent UCI Informatics grad.

## Run locally

```bash
npm install
npm run dev
```

Then open http://localhost:5173.

## Build

```bash
npm run build      # outputs to dist/
npm run preview    # serve the production build
```

## Files

- `src/App.jsx` — the page, top-to-bottom
- `src/data.js` — projects, skills, education, writing (edit this to make it yours)
- `src/components/` — Hero, TopBar, ProjectTile, Section blocks
- `src/index.css` — global base
